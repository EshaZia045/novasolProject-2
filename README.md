 NovaSol Mobile App Development – Milestone 2

## 📱 Project Overview

This repository contains my submission for **NovaSol Software Engineering Internship – Mobile Track, Milestone 2**.

The project demonstrates React Native development using Expo by completing three practical tasks:

- Debugging an existing playlist application
- Building a Pomodoro productivity timer
- Implementing a GPS-based location check-in application

---

## 🚀 Features

### 1. Music Playlist Bug Fix
- Fixed missing component props
- Resolved infinite rendering loop
- Added unique keys for rendered list items
- Improved application stability

### 2. Pomodoro Timer
- 25-minute countdown timer
- Start functionality
- Pause functionality
- Reset functionality
- MM:SS time format
- Automatic countdown using `setInterval`
- Proper cleanup with React Hooks

### 3. Remote Worker Location Check-In
- Requests foreground location permission
- Retrieves current GPS coordinates
- Displays latitude and longitude
- Shows successful check-in status
- Uses Expo Location API
